# Natours - Tour & Travel Landing Page 🌍✈️

A beautiful and responsive **Tour & Travel Landing Page** built using **HTML5** and **CSS3**.  
This project showcases modern UI/UX design with animations, feature cards, tour packages, customer stories, and immersive visuals inspired by nature and adventure travel websites.

---

## 🚀 Features

- Fully Responsive Layout
- Modern Hero Section
- Smooth UI Design
- Travel Tour Cards
- Hover Effects & Animations
- Customer Stories Section
- Background Video Integration
- Nature-inspired Color Theme
- Clean & Structured Code

---

## 🛠️ Built With

- HTML5
- CSS3
- Google Fonts
- Custom Icons
